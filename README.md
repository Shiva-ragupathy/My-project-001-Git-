# My Project 001 – Git Practice Repository

This repository is created to practice Git and GitHub workflows using 
VS Code connected to a remote Ubuntu server via SSH.

## What I Have Achieved

- Initialized a Git repository
- Created and managed branches
- Performed merge and rebase
- Used git stash
- Pushed branches to GitHub
- Created and merged Pull Requests

## Tools Used
- Ubuntu
- Git
- GitHub
- VS Code (Remote SSH)

This project demonstrates hands-on version control practice.
